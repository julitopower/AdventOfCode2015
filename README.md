## Introduction

This projects solves the problems of the Advent of Code 2015 and 2017 (contrary to what the name may suggest), in C++11.

http://adventofcode.com/

## How to build

### Requisites

* CMake >= 2.8
* Boost >= 1.40 (only for 2015)
* OpenSSL (only for 2015)

### 2015

On the root folder

```shell
mkdir build
cd build
cmake ..
make
```


### 2017

```shell
cd 2017
mkdir build
cd build
cmake ..
make
```
